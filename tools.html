6.1.2 What is it?

Jason is an app that lets you bookmark and run multiple Jasonette apps without building your own. Jason itself is built with Jasonette.

Jasonette is the open source app framework that lets you build your own publishable app.

6.1.4 How to Install

Installing Jason App is similar to creating a custom Jasonette App. The main difference is that you only made this once and you will be ready to start testing your JSONs.

    1. Download Jasonette from the releases repository (https://github.com/jasonelle/jasonelle/releases).
    1. Add the Jason url to the project.
    1. Build it in XCode or Android Studio.

The url you must use is https://raw.githubusercontent.com/jasonelle/docs/develop/examples/jasonette/apps/jason-app/index.json

Jason App in Xcode

The complete source code is in the following directory https://github.com/jasonelle/docs/tree/develop/examples/jasonette/apps/jason-app
